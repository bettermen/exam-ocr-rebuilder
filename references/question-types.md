# 题型分类标准

## 题型代码与识别规则

### 1. 单项选择题 (single_choice)

**识别特征**：
- 题干后跟 A/B/C/D（通常4个选项，有时5-6个）
- 题干中可能出现"下列""正确的是""不正确的是"等关键词
- 答案为单个字母

**示例**：
```
1. 下列哪个选项是正确的？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：B
```

**正则模式**：
- 题号：`^\d+[\.．、\)]\s*`
- 选项：`^[A-Z][\.．、\)]\s*`
- 答案：`答案[:：]\s*[A-Z]`

---

### 2. 多项选择题 (multi_choice)

**识别特征**：
- 选项通常为 A/B/C/D/E（4-6个）
- 题干中有"多选""至少有两个""不正确的是"等关键词
- 答案为多个字母（如"ABD"、"ABC"）

**与单选题区分**：
- 题干包含"多选""多选""选出所有"等关键词
- 答案为多个字母
- 大题说明中有"多项选择"字样

---

### 3. 判断题 (true_false)

**识别特征**：
- 题干为陈述句
- 答案为"对/错""正确/错误""T/F""√/×"
- 题干中常有"判断...是否正确"

**正则模式**：
- 答案：`答案[:：]\s*(对|错|正确|错误|T|F|√|×)`

---

### 4. 填空题 (fill_blank)

**识别特征**：
- 题干中包含下划线 `___` 或空括号 `（）` `()`
- 通常为不完整句子，需要填写缺失部分
- 一个题目可能包含多个空

**正则模式**：
- 空格：`_{2,}|（\s*）|\(\s*\)|〔\s*〕`
- 答案：`答案[:：]\s*(.+)`，可能有多个值用逗号或顿号分隔

---

### 5. 简答题 (short_answer)

**识别特征**：
- 题干以"简述""简要说明""简要分析""列举"开头
- 需要文字回答，但不需要长篇论述
- 答案通常为几句话

---

### 6. 计算题 (calculation)

**识别特征**：
- 题干以"计算""求解""求"开头
- 通常涉及数学公式、物理计算
- 答案包含计算过程和结果
- 常含数学符号：∫、∑、√、²、³等

**公式处理**：
- OCR识别的数学公式转为LaTeX
- 上标：x² → $x^2$
- 下标：a₁ → $a_1$
- 分数：a/b → $\frac{a}{b}$
- 根号：√x → $\sqrt{x}$
- 积分：∫ → $\int$
- 求和：∑ → $\sum$

---

### 7. 论述题 (essay)

**识别特征**：
- 题干以"论述""分析""谈谈""阐述""试论"开头
- 需要长篇文字回答
- 答案通常为多段文字

---

### 8. 阅读理解 (reading)

**识别特征**：
- 包含一段材料（文章/诗歌/文言文等）
- 材料后跟多个小题
- 小题题号通常为 1.1, 1.2 或 (1)(2)(3) 或 ①②③

**结构**：
```json
{
  "type": "reading",
  "material": "阅读材料全文...",
  "questions": [
    {"sub_id": 1, "stem": "小题题干", "answer": "答案"},
    {"sub_id": 2, "stem": "小题题干", "answer": "答案"}
  ]
}
```

---

### 9. 完形填空 (cloze)

**识别特征**：
- 一篇文章中有编号空格（如 (1)___, (2)___ 或 21.___, 22.___）
- 每个空格对应一个选项组
- 常见于英语试卷

**结构**：
```json
{
  "type": "cloze",
  "material": "完形填空文章（含编号空格标记）...",
  "questions": [
    {"sub_id": 21, "options": {"A": "...", "B": "...", "C": "...", "D": "..."}, "answer": "A"},
    {"sub_id": 22, "options": {"A": "...", "B": "...", "C": "...", "D": "..."}, "answer": "C"}
  ]
}
```

---

### 10. 证明题 (proof)

**识别特征**：
- 题干以"证明""求证"开头
- 通常含几何图形或数学条件
- 答案为证明过程

---

### 11. 作图题 (drawing)

**识别特征**：
- 题干以"作图""画出""绘制"开头
- 需要在坐标系或图纸上作图
- 答案为图像或作图步骤

**处理**：题目中引用的图片需单独提取保存。

---

### 12. 其他题型 (other)

不属于以上类型的题目，按原始文本保存。

---

## 分值提取规则

| 格式 | 正则 | 示例 |
|------|------|------|
| 题后分值 | `[（(]\s*\d+\s*分\s*[）)]` | （2分）、(3分) |
| 大题说明 | `每[小]?题\s*\d+\s*分` | 每小题2分 |
| 大题总分 | `共\s*\d+\s*分` | 共40分 |
| 题首分值 | `^\d+[\.．、]\s*\[?\d+\s*分\]?` | 1.[5分] |

## 答案分离规则

1. **答案在题目后**：用"答案："或"【答案】"标记，直接提取
2. **答案在试卷末尾**：单独的"参考答案"部分，按题号匹配
3. **无答案**：answer字段设为null
4. **答案含解析**：answer存答案，explanation存解析
