# 试卷OCR重建助手

## 功能概述

专业的试卷数字化工程师 Skill，将 PDF 或图片格式的试卷通过 OCR 识别 + AI 结构化解析 + LLM 审核校正，重建为清晰、可编辑的新试卷。

## 核心能力

- **PDF/图片识别**：支持 PDF 文件和图片（jpg/png/bmp）输入
- **OCR 识别**：优先使用 tencent-docs OCR（高精度模式），备选 PaddleOCR
- **题型分类**：自动识别 11 种题型（单选/多选/判断/填空/简答/计算/论述/阅读/完形/证明/作图）
- **公式转换**：自动将数学公式转为 LaTeX 格式
- **LLM 审核**：8 大维度审核校正 OCR 识别错误
- **多格式输出**：HTML 交互报告 + JSON 结构数据 + Markdown 试卷

## 使用场景

1. **教师数字化试卷库**：将纸质试卷转为可编辑电子版
2. **学生整理错题**：识别试卷题目，提取错题重构
3. **教育机构内容迁移**：批量处理历史试卷，建立数字化题库
4. **在线考试系统**：将传统试卷转为在线考试平台格式

## 处理流程

```
输入（PDF/图片）
  ↓
[第一步] PDF 转图片（如需要）
  ↓
[第二步] OCR 文字识别
  ↓
[第三步] AI 结构化解析（题型分类、题干/选项/答案提取、LaTeX 公式转换）
  ↓
[第四步] LLM 大模型审核检查（校正 OCR 错误、补全截断内容、修正 LaTeX 公式）
  ↓
[第五步] 生成输出（HTML 交互报告 + JSON + Markdown）
  ↓
[第六步] （可选）创建腾讯文档在线编辑版
```

## 审核维度

| 维度 | 检查内容 |
|------|-----------|
| 题干完整性 | 是否截断、缺字 |
| 字符错误 | 错别字、同音字 |
| 数字错误 | 数字识别错误（0↔O、1↔l） |
| 公式 LaTeX | 公式是否准确 |
| 选项完整性 | 选项是否缺失 |
| 题号连续性 | 是否跳号、重复 |
| 答案正确性 | 答案是否匹配 |
| 题型一致性 | 题型分类是否正确 |

## 输出格式

### 1. HTML 交互报告
- 题型标签筛选
- 答案/解析显示隐藏开关
- KaTeX 数学公式渲染
- 审核报告面板（右下角悬浮按钮）
- 响应式布局（手机可看）

### 2. JSON 结构数据

## 依赖要求

- **Python 3.8+**（PDF 转图片功能）
- **PyMuPDF**（fitz，PDF 处理）
- **tencent-docs 连接器**（OCR 识别，已连接优先使用）

## 示例

输入：一份 5 页的数学期中试卷 PDF

输出：
- `exam_report.html` - 交互式试卷报告（含答案、解析、审核状态）
- `exam_data.json` - 结构化试题数据（可导入题库系统）
- `exam_paper.md` - Markdown 格式试卷（可编辑、可打印）

## 限制说明

- OCR 识别准确率取决于原始试卷清晰度（建议 DPI ≥ 300）
- 手写试卷识别效果较差（仅支持印刷体）
- 复杂表格题型可能需要人工复核
- LLM 审核使用当前会话模型，不增加额外 API 成本
